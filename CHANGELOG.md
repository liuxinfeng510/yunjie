# 更新日志

## v1.5.0 (2026-03-16)

### 新增功能
- **批量导入功能**: 会员列表、药品列表支持批量Excel导入，自动匹配字段并可手动调整映射
- **批量导入对话框**: 三步向导式操作（上传 -> 字段映射 -> 导入结果），支持数据预览
- **药品原编号字段**: 药品表新增 `original_code` 字段，用于关联旧系统商品编号，支持库存数据迁移
- **库存导入增强**: 库存导入支持通过原编号、药品名称等多种方式匹配药品（drugCode -> barcode -> originalCode -> drugName -> genericName）
- **跳过日志功能**: 数据迁移任务新增跳过原因分类追踪，按类别统计跳过数量及示例行号
- **中药知识库管理**: 新增中药材知识库的增删改查和搜索功能
- **药品知识库管理**: 新增西药/成药知识库的增删改查和搜索功能
- **中药配伍禁忌管理**: 新增配伍禁忌的增删改查和批量管理功能

### 功能优化
- **数据迁移字段自动匹配**: 使用 EasyExcel AnalysisEventListener 自动检测Excel表头，通过别名映射自动匹配系统字段
- **会员导入姓名解析**: 支持从 "姓名 (性别)" 格式中自动提取姓名和性别信息
- **POS会员搜索**: 多个匹配结果时显示下拉列表供选择，显示姓名、电话、积分信息
- **下拉框快照保存**: 表单中的下拉框选择内容自动保存到快照，确保历史数据一致性
- **导入执行反馈**: 导入执行后显示结果对话框，包含总数/成功/跳过/失败的统计和详细错误信息

### 数据库迁移
- V22: 生产企业种子数据
- V23-V24: 中药知识库补充和新品种
- V25-V31: 药品知识库（感冒/抗生素/心脑血管/消化/呼吸/神经/骨科/维生素/中成药/儿科/皮肤/五官/泌尿/妇科/抗过敏）
- V32: 下拉框快照字段
- V33: 数据迁移跳过日志字段
- V34: 药品原系统编号字段及索引

### 涉及模块
- 后端: DrugController, MemberController, DataMigrationService, BatchImportService, 11个实体类快照字段, 8个Service快照保存逻辑
- 前端: DrugList, MemberList, POS, DataMigration, BatchImportDialog, DrugKnowledge, HerbKnowledge, Incompatibility
- 数据库: V22 ~ V34 共13个迁移脚本
